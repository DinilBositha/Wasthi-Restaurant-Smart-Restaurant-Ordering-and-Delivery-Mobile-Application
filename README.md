### 🚀 Key Features

* **Modern UI/UX** – Clean and interactive design implemented using Java and XML.
* **Firebase Integration** – Real-time data synchronization and secure authentication.
* **Intuitive Navigation** – Seamless user experience with smooth transitions.
* **Optimized Performance** – Fast loading times and efficient memory management.

| **IDE** | Android Studio |
